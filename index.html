Windows Native API Reference

WinAPI → Nt* 함수 대응 정보 조회

WinAPI 호출과 대응되는 Nt* 함수, 관련 구조체 및 주의사항을 한 화면에서 확인할 수 있는 Windows 저수준 개발용 인터페이스입니다.

작업 화면

입력

분석할 WinAPI 함수 또는 코드 스니펫을 입력하세요.

파일, 메모리, 프로세스 관련 호출 입력 가능

결과

대응 Nt* 함수와 구조체, 반환값 정보를 표시합니다.

대응 Nt* 함수

NtCreateFile NtReadFile NtAllocateVirtualMemory NtCreateUserProcess

함수 매핑

WinAPI Nt* 설명
CreateFileW NtCreateFile 파일 생성 및 접근 처리
ReadFile NtReadFile I/O 요청 기반 읽기 처리
VirtualAlloc NtAllocateVirtualMemory 가상 메모리 할당 처리
CreateProcessW NtCreateUserProcess 프로세스 생성 처리

참고 사항

  • 구조체 사용: 일부 Nt* 함수는 별도의 Native 구조체 사용이 필요합니다.
  • 반환값: Win32 오류 코드 대신 NTSTATUS 값을 반환합니다.
  • 호환성: Windows 버전에 따라 내부 동작이 달라질 수 있습니다.
